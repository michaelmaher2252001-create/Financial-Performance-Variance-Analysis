# Financial Performance & Variance Analysis Dashboard

## Overview
This project evaluates financial performance by comparing actual sales against targets,
analyzing variances, profitability, and country-level performance.

## Business Problem
Management requires accurate tracking of financial performance to identify gaps
between actual results and targets.

## Data Overview
- Sales
- Target values
- Profit
- Product data
- Country

## Tools Used
- Power BI
- DAX
- Excel

## Analysis Performed
- Actual vs target sales comparison
- Variance and variance percentage analysis
- Profitability by country
- Product-level contribution analysis

## Key Insights
- Overall sales slightly missed targets with a negative variance
- Certain countries drive the majority of profit
- Discount strategies impact profitability significantly

## Business Value
Helps management monitor performance,
optimize strategy, and improve financial decision-making.
